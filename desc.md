# Репозиторій студентських ШІ-агентів на платформі n8n

Цей репозиторій містить роботи студентів кафедри математичного моделювання і аналізу даних, створені в межах курсу **«Штучний інтелект для відновлення країни»**, що спрямований на розробку прикладних ШІ-агентів на платформі **n8n**.

У репозиторії зібрано вихідні *JSON-файли* робочих процесів та *відеопрезентації* таких систем:

- **ШІ-агент для перевірки наявності супутникових знімків**  
  *Автор: Олександр Матушевський*

- **ШІ-агент для отримання даних про NDVI для фіксованої території**  
  *Автори: Емма Бродіна, Тимофй Мельник*

- **ШІ-агент для отримання даних про NDVI для довільних територій**  
  *Автори: Максим Намчук, Ольга Подвисоцька*

- **ШІ-агент-помічник на тематику Sustainable Development**  
  *Автор: Вадим Чужда*

- **ШІ-агент-помічник для відвідувачів сайту кафедри та абітурієнтів**  
  *Автор: Гліб Стрілець*

- **ШІ-агент для аналізу російського YouTube-контенту на виявлення пропаганди**  
  *Автори: Ігор Вітенко, Дмитро Харь*

- **ШІ-агент для моніторингу новин та виявлення випадків розливу нафти в Чорному морі**  
  *Автори: Максим Нагорський, Яков Бондарець*

- **ШІ-агент для рекомендацій підрядників з фокусом на малий та середній бізнес**  
  *Автори: Олександр Кравчук, Іван Щербина*
